Quick Start

# Install dependencies

# Build the parser
npm run build

# Run basic example
node examples/basic-parser.js

# Start web server for browser examples
cd examples && python3 -m http.server 8000
# Then visit http://localhost:8000

Supported Messages

The parser handles all standard MAVLink messages including:

  • HEARTBEAT - System status and keep-alive
  • GPS_RAW_INT - Raw GPS position data
  • ATTITUDE - Vehicle orientation (roll, pitch, yaw)
  • GLOBAL_POSITION_INT - Global position with altitude
  • VFR_HUD - HUD display data
  • MISSION_ITEM - Waypoint data
  • PARAM_VALUE - Parameter values
  • And many more...
